# 老几测试 500 个真实用户冲刺

这是微信小程序 **老几测试** 的公开拉新页项目。

目标很简单：帮助小程序拿到第一批 **500 个真实用户**，满足微信小程序流量主广告位开通前的基础用户门槛。

## 怎么帮忙

1. 打开微信。
2. 搜索：`老几测试`。
3. 进入小程序，完整完成一次测试。
4. 如果愿意，把小程序或这个页面转发给朋友。

## 我们不做什么

- 不刷 IP。
- 不跑代理。
- 不做机器人访问。
- 不诱导分享。
- 不承诺虚假结果。
- 不伪造用户行为。

真实用户比数字更重要。后续要开微信流量主广告位，干净的用户来源比短期刷量更有价值。
